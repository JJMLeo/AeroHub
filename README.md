# AeroHub - 航空工程计算工具

![HarmonyOS](https://img.shields.io/badge/HarmonyOS-4.0-blue.svg)
![ArkUI](https://img.shields.io/badge/ArkUI-3.0-green.svg)
![License](https://img.shields.io/badge/license-Proprietary-lightgrey)

**AeroHub** 是一款基于 HarmonyOS ArkUI 开发的航空工程计算工具应用，专为航空航天工程师、学生和爱好者设计。应用提供全面的航空大气参数计算、热力学属性分析和单位换算功能，采用国际标准大气模型（ISA）进行精确计算。

## ✨ 功能特性

### 🚀 气动计算模块
- **输入参数**：高度（km/ft）、温度（K/°C/°F/°R）、速度（马赫数/表速）
- **计算输出**：静压、静温、密度、声速、马赫数、表速、动压、总温、总压、总焓等17项关键航空参数
- **计算模型**：基于国际标准大气模型（ISA）的精确计算
- **历史记录**：支持最多20条计算历史保存

### 🌡️ 标准大气计算模块
- **标准大气模型**：计算不同高度下的标准大气温度、压力、密度
- **高度范围**：支持0-80km高度范围计算
- **多层模型**：对流层、平流层、中间层分层计算

### ⚛️ 物性参数计算模块
- **热力学属性**：计算空气的比热比、气体常数、摩尔质量、定压比热容、定容比热容
- **状态参数**：计算比焓、熵函数等热力学状态参数
- **温度范围**：支持200K-2000K温度范围计算

### 🔄 单位换算模块
- **支持单位**：长度、速度、压力、温度、密度、力、能量、功率、角度等9类物理量
- **换算公式**：实时显示所有单位间的换算关系
- **智能交互**：
  - 左侧为主输入端，改变左侧单位时保持左侧数值不变
  - 改变右侧单位时同样保持左侧数值不变
  - 输入框点击清空，延迟刷新机制优化用户体验
- **结果展示**：数值右对齐、单位左对齐的清晰显示格式

## 🚀 安装与构建

### 环境要求
- **DevEco Studio**: 4.0 或更高版本
- **HarmonyOS SDK**: API 9 或更高版本
- **Node.js**: 18.0 或更高版本

### 构建步骤
1. 克隆项目到本地
2. 使用 DevEco Studio 打开项目
3. 等待依赖自动下载完成
4. 点击工具栏中的 **Build** → **Build HAP(s)** 进行构建
5. 构建完成后，在 `entry/build/default/outputs/default` 目录下找到生成的 HAP 文件

### 运行调试
1. 连接 HarmonyOS 设备或启动模拟器
2. 点击工具栏中的 **Run** → **Run 'entry'**
3. 应用将自动安装并运行在目标设备上

## 📱 使用说明

### 主页面
- 显示四个功能模块的入口卡片
- 点击卡片跳转到对应功能页面
- 底部显示版本信息和版权声明

### 气动计算页面
1. **输入参数**：
   - 高度：选择单位（km/ft）并输入数值
   - 温度：选择温度类型（大气温度/温度偏差）和单位（K/°C/°F/°R）
   - 速度：选择速度类型（马赫数/表速）并输入数值

2. **计算结果**：
   - 点击"计算"按钮后，下方显示17项计算结果
   - 结果按类别分组显示，支持滚动查看

3. **历史记录**：
   - 每次计算结果自动保存到历史记录
   - 最多保存20条记录，点击历史记录可快速回填参数

### 标准大气计算页面
1. **输入高度**：输入高度值（km）
2. **计算结果**：自动计算并显示该高度下的标准大气温度、压力、密度
3. **物理常数**：显示气体常数、通用气体常数、空气摩尔质量等常数

### 物性参数计算页面
1. **输入参数**：输入温度（K）和油气比（FAR）
2. **计算结果**：计算并显示空气的热力学属性
3. **参数说明**：每个参数都有详细的单位说明

### 单位换算页面
1. **选择物理量**：从9类物理量中选择需要换算的类型
2. **输入数值**：在左侧输入框输入数值
3. **选择单位**：左右两侧分别选择源单位和目标单位
4. **查看结果**：
   - 下方显示所有单位的换算结果
   - 中间区域显示换算公式
   - 历史记录显示最近的换算操作

## 🧮 计算原理

### 国际标准大气模型（ISA）
应用采用国际标准大气模型进行计算，模型分为三层：

1. **对流层（0-11km）**：
   - 温度梯度：-6.5°C/km
   - 压力公式：P = P₀(1 - H/44.308)^5.25588

2. **平流层下层（11-24.994km）**：
   - 温度恒定：-56.5°C
   - 压力公式：P = P₁₁·exp((11 - H)/6.338)

3. **平流层上层（>24.994km）**：
   - 温度梯度：+0.0029892°C/km
   - 压力公式：P = P₂₅·(216.65/T)^11.8

### 热力学计算
- **气体属性计算**：基于多项式拟合方法，计算空气在200K-2000K温度范围内的热力学属性
- **空气成分**：考虑干空气和燃气的混合，支持0-2%的油气比范围
- **物理常数**：
  - 通用气体常数：R_univ = 8.314462618 J/(mol·K)
  - 空气摩尔质量：28.9647 g/mol（干空气）

### 单位换算算法
- **长度单位**：基于国际单位制与英制单位的精确换算系数
- **温度单位**：区分绝对温度和温差的不同换算规则
- **压力单位**：支持Pa、kPa、atm、psi等多种压力单位转换
- **实时计算**：采用左侧为主输入端的交互逻辑，确保计算一致性

## 🐛 已知问题与限制

### 计算精度
- 温度范围限制在200K-2000K之间，超出范围使用边界值
- 油气比范围限制在0-2%之间
- 高度计算在极端条件下可能存在微小误差

### 设备兼容性
- 需要 HarmonyOS 4.0 或更高版本
- 部分高级功能需要设备支持相应的计算能力

### 功能限制
- 目前仅支持空气的热力学计算
- 单位换算不支持自定义换算系数
